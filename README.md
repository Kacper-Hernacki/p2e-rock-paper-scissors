# React app beginner friendly tutorial

Welcome to the repo with code of a simple rock paper scissor react game.

It includes:

- basic react setup,
- implementation of hooks,
- react-router-dom,
- react-redux,
- styled components.

## Setup

1. Clone the repo.
2. Change directory to an app by:
   `cd P2E-RPS`
3. Install dependencies:
   `npm i`
4. Run an app:
   `npm start`
